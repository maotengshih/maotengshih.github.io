Hexo 部落格用法指南

發表於 2023-08-31 更新於 2023-09-08 分類於 大分類 ， 小分類 ， 小分類

Hexo 部落格

清快取 接著執行 編譯加佈署

hexo clean && hexo g –d

快速啟動

建立新文章

$ hexo new article-name-here

啟動本地伺服器

$ hexo serve

指定 port

hexo s -p 4001

編譯靜態網頁

$ hexo generate

佈署至 github (需先設定好config)

$ hexo deploy

寫作

建立草稿

hexo new draft new-draft-name

建立文章

hexo new post new-post-name

建立頁面 http://localhost:4000/new-page-name/

hexo new page new-page-name

檔名變更 （ new_post_name setting in _config.yml ）

:year-:month-:day-:title.md

將草稿匣的文章移動至已發佈

hexo publish post test-article

從預先搭好的鷹架建立你的文章 ( 在 scaffolds 資料夾中有預先建立好的模版 )

hexo [layout] post-name

Front-matter

特殊引用

Writing is good, writing is cool,

Author url link-title

Youtube 引用

Youtube 播放清單引用

一鍵發佈，兩種寫法

$ hexo generate –deploy
$ hexo deploy –generate

串接寫法&& 與簡寫

hexo clean && hexo g –d
