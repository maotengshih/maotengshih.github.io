  • 操作數值(value)：20 + (5 * 8) + (8/3)
  • 操作字串(string)：print("My name is " + "John")

條件式(conditional)：

邏輯運算子(operator):

  • ==：兩邊一樣則為 true
  • !=：兩邊不一樣則為 true
  • >：左大於右則為 true
  • <：右大於左則為 true
  • >=：左大於等於右則為 true
  • <=：右大於等於左則為 true

  • windows 作業系統為例，常用的指令如下：

指令 說明
cd 切換目錄
man 指令使用手冊
cd .. 往上一層目錄移動
cd -al 往上一層目錄移動
pwd 取得目前所在位置
ls 列出目前檔案列表
touch 新增檔案
ehco 印出你所打的東西
ehco index.html 新增 index.html 檔案

重新導向
ehco “Hello” > index.html 新增內容為 Hello 的 index.html 檔案

新增並取代內容
echo “append to last” >> index.html 新增內容到檔案

cat 印出檔案內容
clear 清除畫面上的內容
grep 抓取關鍵字
wget 下載檔案
curl 送出 request
(pipe) 將左邊指令的輸出作為右邊指令的輸入
