CodingDay002：CSS 基礎操作與綜合演練

發表於 2023-10-26

昨天我學到使用 html 標記網頁內容，今天學習的是使用 CSS 來將網頁內容進行排版。

  1. 首先在 <head> 內增加 <link href="styles.css" rel="stylesheet"/> 使瀏覽器知道透過參照連結去找到所屬的樣式表。

  2. 在 styles.css 內透過選擇器選取目標物

    • type selector : elementtype {}
    • id selector : #idname {}
    • class selector : .classname {}

  3. 對目標物進行樣式變更

    • 背景圖片 : background-image: url();
    • 文字對齊方式：text-align: center;
    • 寬度設定：`width: 80%;
    • 底色設定：background-color: burlywood;
    • 留邊設定：`margin-left: auto;

  4. 綜合演練： 在同一列顯示品名與價格

    • 將每一個品項以 <article> 群組化，並設定其 classname=item
    • 將單一品項的產品名稱、價格以 <p class=product> <p class=price> 命名
    • 透過選擇器選取 .item p {display: inline-block;}
    • 顯示為行內元素：display: inline-block;
    • 寬度設定：width: 50%;
